# Решение проблем

## Проект не запускается

### 1. Проверьте установку зависимостей
```bash
npm install
```

### 2. Проверьте, что сервер запущен
```bash
npm run dev
```

Сервер должен запуститься на `http://localhost:3000` (или другом порту, если 3000 занят)

### 3. Откройте браузер и проверьте консоль

Откройте DevTools (F12) и проверьте вкладку Console на наличие ошибок.

### 4. Типичные проблемы

#### Ошибка: "Cannot find module"
- Убедитесь, что все файлы скопированы в папку `src/`
- Проверьте пути импортов в файлах

#### Ошибка: "Failed to resolve import"
- Проверьте, что все зависимости установлены: `npm install`
- Убедитесь, что пути к файлам правильные (с расширениями .js)

#### Белый экран / ничего не отображается
- Проверьте консоль браузера на ошибки
- Убедитесь, что элемент `#content` существует в `index.html`
- Проверьте, что CSS файлы загружаются (вкладка Network в DevTools)

#### Ошибки с Vue Router
- Убедитесь, что используется Vue Router 4
- Проверьте синтаксис роутов в `src/app/router.js`

### 5. Проверка структуры проекта

Убедитесь, что структура выглядит так:
```
├── src/
│   ├── app/
│   │   ├── pages/
│   │   ├── widgets/
│   │   ├── app.js
│   │   └── router.js
│   ├── assets/
│   │   ├── css/
│   │   ├── fonts/
│   │   └── images/
│   └── main.js
├── index.html
├── vite.config.js
└── package.json
```

### 6. Очистка и переустановка

Если ничего не помогает:
```bash
# Удалите node_modules и package-lock.json
rm -rf node_modules package-lock.json

# Переустановите зависимости
npm install

# Запустите снова
npm run dev
```

### 7. Проверка версий

Убедитесь, что установлены правильные версии:
- Node.js: 16+ (проверьте: `node --version`)
- npm: 7+ (проверьте: `npm --version`)
