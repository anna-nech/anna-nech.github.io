# Affiliate Dashboard

Проект мигрирован с CDN/Global Vue на Vite + Vue 3.

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск проекта

### Режим разработки
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

### Сборка для продакшена
```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

### Просмотр собранного проекта
```bash
npm run preview
```

## Структура проекта

```
├── src/
│   ├── app/
│   │   ├── pages/          # Страницы приложения
│   │   ├── widgets/        # Виджеты/компоненты
│   │   ├── app.js          # Главный файл приложения
│   │   └── router.js       # Конфигурация роутера
│   ├── assets/
│   │   ├── css/            # Стили
│   │   ├── fonts/          # Шрифты
│   │   └── images/         # Изображения
│   └── main.js             # Точка входа приложения
├── index.html              # Главный HTML файл
├── vite.config.js          # Конфигурация Vite
└── package.json            # Зависимости проекта
```

## Изменения при миграции

1. **Vue 3**: Проект использует Vue 3 вместо Vue 2
2. **Vue Router 4**: Обновлен до версии 4
3. **Vite**: Используется Vite вместо CDN для сборки и разработки
4. **ES Modules**: Все файлы используют ES модули
5. **Импорты**: Vue и Vue Router импортируются как модули вместо глобальных переменных

## Зависимости

- **vue**: ^3.4.21
- **vue-router**: ^4.3.0
- **axios**: ^1.6.7
- **chart.js**: ^4.4.2

## Разработка

Все исходные файлы находятся в папке `src/`. При изменении файлов Vite автоматически перезагрузит страницу в браузере (HMR - Hot Module Replacement).
